

There are 2 directories for program 2: prog2server and prog2client, each with their own makefile. Open separate terminals for each.

1. In prog2server directory, use command:

	make
   
   to compile the prog 2 server files. Then use command:

	./prog2server.out

   to run the server.


2. In prog2client directory, use command:

	make
   
   to compile the prog 2 client files. Then use command:

	./prog2client.out

   to run the client.
